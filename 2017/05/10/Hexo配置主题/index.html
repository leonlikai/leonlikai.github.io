Hexo 配置主题

Hexo主题存放路径

~/hexo/themes/

主题配置步骤


  1. 下载主题，进入hexo目录下，用git bash执行（最后一个参数为路径名）：


    git clone https://github.com/iissnan/hexo-theme-next themes/next


  2. 启用主题，修改hexo的配置文件_config.yml，将theme设置为next

  3. 测试、发布
