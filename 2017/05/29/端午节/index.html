端午节

端午节名称

古语中，端是初的意思，午与五同义，五为阳数（奇数为阳数），因此端午即初五，端午节又叫端阳节。

端午节的来历


  1. 屈原投江说

  2. 纪念介子推说

  3. 伍子胥死谏吴王夫差说

  4. 曹娥投江寻父说

端午节活动

  • 赛龙舟
  • 包粽子
  • 栓五色丝线、挂荷包、佩香囊
  • 喝雄黄酒
  • 挂菖蒲、蒿草、艾叶等

端午节祝福语

端午节祝福语用安康，从端午节的来历看，基本上都是死人的事，从端午节活动上来看，挂五色丝线、喝雄黄酒、挂艾草等都是为了驱害虫，因此，端午是一个凶日，至少不是喜庆的节日。
端午节带着一种期盼生活安康的意愿，因此，端午祝福语用安康最合适不过了。
