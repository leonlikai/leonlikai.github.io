win7环境下用github和hexo搭建静态博客

这是我的第一篇Hexo博客文章。

今天研究了在windows环境下用Github和hexo搭建静态博客，也是第一次正式使用github.


使用Github和Hexo搭建静态博客相当简单，网上有大量教程，照着一步步做即可。详细步骤不再赘述了，这里把使用心得说一下。

###Git和Hexo环境搭建
Git为github所用的分布式管理系统，类似svn工具。
Hexo为静态博客生成器，用node.js开发。
环境搭建需要的工作有：


  1. 安装上述工具，git，node.js，hexo。

  2. 注册github用户；

  3. 生成并增加ssh key，ssh key用于通过ssh向github仓库更新版本时使用。我的理解是ssh key是公钥，因为ssh是加密传输的，github用的是rsa加密算法，因此生成一个公钥来确保加密。

  4. hexo安装、初始化、部署。hexo可以在本地起服务测试。

  5. 注册github repository

  6. 将本地hexo项目托管到github。在_config.yml下配置仓库地址等信息，安装hexo-deployer-git插件。

  7. 部署本地主题到github上。

  8. 更新内容后重新发布

###发布博文

    用markdown编辑博文，生成md文件，放入_posts目录下，然后发布。这里发布成功了，但是markdown的“###”格式hexo不认(本地和服务器上均不认)，原因待确认。
